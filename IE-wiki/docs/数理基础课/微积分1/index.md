# 微积分Ⅰ

## 课程学习内容

微积分是工科同学重要的工具，也是未来进一步学习其它数学课的不可或缺的基础。微积分Ⅰ重点在于一元函数的极限与连续，导数及其应用（微分中值定理、泰勒展开等）以及积分学（不定积分、定积分、黎曼可积性、广义积分等）、解析几何的基本内容。
