## 在南大使用Git

南大为所有同学提供了[GitLab](https://git.nju.edu.cn/) Premium版本的账号，拥有超大容量的存储空间。

Github的服务有时会被墙（git clone, git push, git pull），你可以选择：

1. 把需要的Github项目导入GitLab，然后在GitLab操作。
2. 翻墙，并配置好终端的网络环境。
3. 在浏览器下载整个项目的压缩包，再在终端解压。
