# 工程经济学

## 课程学习内容

工程经济学是工程与管理学科的交叉领域，主要研究如何在工程项目中经济合理地分配有限资源，核心目标是为工程决策提供经济分析依据。具体内容可参考[在线教学平台](https://etextnju.cn/)。

## 任课老师

程书萍老师

## 课程亮点

为了减轻同学们的负担，程老师建立了一个[在线教学平台](https://etextnju.cn/)，包含自编教材、金融计算器、交互式项目评价系统，将复杂的计算过程交给计算机去处理。课程讲授方式与传统教材有很大区别，更加轻松。

## 考核方式

这里介绍25年的情况，之后考核方式可能会改变。考核包含：
- 期中和期末两次测验，都是在课堂时间进行，不会在期末周组织统一的考试。考试为开卷，可以查阅任何资料（包括互联网和大模型）。
- 使用教学平台的交互式项目评价系统进行一次项目分析，提交PDF文件（实际上就是在网站上填空）。

期中考试没有提前通知，占总分30%。

看到这里，你也清楚了，25年的考核非常宽松，即使上课不认真听，对考核也不会有太大影响，但是如果翘课错过考试，会收获一份尴尬且印象深刻的回忆。

## 学习建议

如果你对工程经济学感兴趣，请认真听讲。如果不感兴趣，可以把精力分到更重要的地方，对最终考核影响不大。
