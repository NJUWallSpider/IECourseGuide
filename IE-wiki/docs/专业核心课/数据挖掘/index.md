# 数据挖掘

## 课程大纲

1. 导论：讲述本课程在课程体系中的重要性、本课程架构、教材资料，以及课程基本要求。数据挖掘基本概念、工具、方法与标准流程
2. 数据清洗：跨行业数据挖掘标准流程/降维、集成、转换与规约
3. 数据挖掘标准流程与模型
4. 处理缺失数据高级方法与案例展示
5. 聚类与异常值探测模型与案例展示
6. 决策树模型的理论模型与算法解析
7. 决策树模型金融公司客户贷款信用评估案例展示
8. 关联规则的理论模型解析与超市产品销售案例展示
9. 主成分分析的理论模型解析与证券市场产品归类案例展示
10. 因子分析的理论模型解析与证券市场产品归类案例展示
11. 逻辑回归多元计量模型与商业银行产品交叉销售案例展示
12. 神经网络预测模型解析与促销活动客户响应案例展示
13. 社会网络聚类模型解析与基金组合最优配置案例展示
14. 结构化方程模型解析：探索性因子分析与验证性因子分析
15. 结构化方程案例展示：证券市场内幕交易的行为动机分析
16. 计算实验金融模型解析与算法交易行为与市场影响案例展示
17. 大作业汇报：根据所学数据挖掘模型与方法，针对给出的理论与现实问题，进行建模、编程、撰写Word版报告。

## 考核方式

虽然课件里说有期中考试，但是2025年实际没有考试，只有最后的期末大作业（具体作业要求见资料包）。

期末大作业实际与上课讲的内容关联不大，即使从开学直接穿越到期末也可以完成作业。

给分比较宽松。
