# 制造与服务系统分析

## 课程学习内容

1. 系统观与基础认知 (Introduction)
    - 什么是系统：建立了“系统”的概念，即由相互联系的部分组成的整体。无论是制造企业（提供有形产品）还是服务企业（提供无形产品），都应作为一个整体来分析。
    - 运营系统的本质：理解运营是将投入（Input）转化为产出（Output）的过程。
2. 运营系统的描述与流程分析 (Process View)流程可视化：
    - 学习如何使用**流程图（Process Flow Charts）**来描述复杂的运营系统。矩形代表活动（Activities），倒三角形代表库存/缓冲（Buffers），箭头代表流向。
    - 生产策略分类：
        - MTS (Make-to-Stock)：备货生产（如麦当劳）。
        - MTO (Make-to-Order)：订单生产（如汉堡王）。
        - ATO (Assemble-to-Order)：组装订单生产。
    - 产品-流程矩阵：理解了不同产量和品种的产品适合什么样的生产流程（如流水线 vs. 零工车间）。
3. 系统绩效度量 (Performance Measures)
    - 如何评价一个系统：如果你不能度量它，就不能管理它。
    - 三大类指标：
        - 财务指标：利润、成本。
        - 外部指标（面向客户）：价格、交付时间、质量、多样性。
        - 内部指标（面向运营能力）：效率、流程时间、灵活性。
    - 核心定律：利特尔定律 (Little's Law) —— 库存 = 产出率 × 流程时间。这是分析系统流量、速度和库存之间关系的最重要工具。
4. 库存管理与分析 (Inventory Analysis)
这是课程中数学模型最集中的部分：
    - 基础管理：库存的定义、ABC分类法（重点管理高价值物品）。
    - 确定性模型：EOQ（经济订货批量）模型。在需求已知的情况下，如何通过权衡“订货成本”和“持有成本”来计算最优订货量。
    - 随机模型：报童模型 (Newsvendor Model)。在需求不确定的情况下（如报纸、生鲜），如何权衡“订多了的浪费”和“订少了的损失”来决定订货量。
    - 供应链协调：提到了收益共享契约 (Revenue Sharing)（以Blockbuster为例），展示了如何通过合同设计实现供应链上下游的双赢。

一句话总结：这门课教你如何用科学的系统思维去拆解企业运营，用流程图看清业务逻辑，用利特尔定律分析效率瓶颈，最后用库存模型（EOQ/报童）在成本与风险之间通过计算做出最优决策。(AI Generated based on slides)
