# 计算机网络

## 授课学院

计算机学院

## 课程学习内容

从网线到应用软件，教你计算机网络是怎么一步步搭建起来的。以太网、互联网、万维网到底是什么东西？为什么我自己的电脑和百度的服务器都在互联网，但是我却没法用手机直接访问电脑？SSH、VPN、内网穿透到底是什么原理？有一台云服务器（公网IP）后，我可以做什么，暴露IP会有哪些风险？如果你想在网络上大显身手，那这门课就是基础。

## 上课指南

计算机网络的课有很多，我上的是秋冬学习苏丰老师开的 3 学分的那个。

课程包含期中考试（开卷）、期末考试（闭卷）、书面作业以及Lab实验。考试内容都来自书面作业。
